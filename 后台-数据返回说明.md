## 返回数据接口定义
### 请求接口
Response Code:200

```python
# 正常情况code为0,detail为空
{
	"code": 0,
	"detail": "",
	"data": {}
}
```

```python
# 业务错误code不为0,detail不为空
{
	"code": 1,
	"detail": "参数错误",
	"data": {}
}
```

### 未登录
Response Code:403
```python
# CSRF验证失败
{"detail":"CSRF Failed: CSRF token missing or incorrect."}
# 无效Token
{"detail":"身份认证信息未提供。"}
# 发生这两个错误都需要用户重新登陆
```

### 未知错误
Response Code:500
```python
{"detail":"unknown error"}
```
